Welcome to PaintCode.

PaintCode is a new helpful tool for your computer, that helps you do daily tasks by simple hand gestures. You can set up to 20 gestures with different tasks like: "Open my browser", "Mark my emails as unread", "Skip song", "Next slide" and much more.

About the developers

PaintCode is developed in the MEET venture lab, a place for alumni of the program to come together and create their own projects.

Open PaintCode

We are planning to create an open API for PaintCode, allowing other developers to use our image-processing technologies.
